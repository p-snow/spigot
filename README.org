A simple socket HTTP server

This is a simple implementation of a Unix Domain Socket server providing an HTTP service that interprets the request into a command string and invokes it on the machine.

* Installation

- building the code
  #+begin_example
  cargo build
  #+end_example
- running the server
  #+begin_example
  cargo run
  #+end_example
